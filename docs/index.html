روش مطالعه برای دفاع

۱. اول صفحه مربوط به خود را بفهمید

در هر بخش، فایل learning.html مفاهیم را ساده توضیح می‌دهد. این فایل برای شروع بهترین است.

۲. بعد نقشه کد را تمرین کنید

فایل code-map.html نشان می‌دهد صفحه از کدام API و کدام route و کدام جدول استفاده می‌کند.

۳. سوال و جواب‌ها را با صدای بلند تمرین کنید

فایل defense-qa.html برای جواب دادن در مقابل استادان آماده شده است.

فرمول طلایی توضیح پروژه

برای هر فیچر این مسیر را بگویید:

Screen -> client/src/api -> Express route -> Service/helper -> MySQL table

مثال: صفحه ایجاد سند در فرانت‌اند، از documentApi.create استفاده می‌کند، route سرور در document.routes.ts اجرا می‌شود، و معلومات در جدول documents ذخیره می‌گردد.
